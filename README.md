# TFG-Super-resolucion-de-imagenes-satelitales
Fine-tuning extremo de SwinIR para Super-resolución de imágenes satelitales.

Un modelo SwinIR sometido a un fine-tuning prolongado (alto número de iteraciones) exclusivamente sobre un conjunto de datos acotado (Región X) desarrollará un 'sobreajuste benigno'. Este sobreajuste le permitirá memorizar las texturas y patrones arquitectónicos únicos de dicha región, logrando una fidelidad de reconstrucción y métricas (PSNR/SSIM) superiores dentro de esa región, superando a cualquier modelo de satélite generalista. Sin embargo, al ser probado en una región geográficamente distinta ('Región Y'), su rendimiento será catastróficamente peor, incluso por debajo de un modelo genérico o de la interpolación bicúbica, probando que el modelo ha 'olvidado' el conocimiento general para memorizar el específico. El proyecto contemplara la generalizacion del modelo frente a imagenes con un zoom y rotacion distintos, y imagenes con nubes que tapen la vision terrestre.
